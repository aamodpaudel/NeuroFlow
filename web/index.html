The Science

How NeuroFlow Works

Two complementary tools working together — one lives in your browser, one on the server.

🎥
LIVE — Browser

MediaPipe Companion

Runs entirely in your browser using your webcam. Monitors browDownLeft + browDownRight blendshapes (AU04 proxy) for real-time furrowing.

if browDown > 0.7 for 3s → .high-load-mode
🔬
AUDIT — Python Server

Py-Feat Auditor

Runs on a local Python server during testing. Precisely tracks AU04 (Brow Furrow) and AU45 (Blink Rate) to generate a Cognitive Load Score.

CLS = (AU04 × 0.7) + (AU45_norm × 0.3)
✅
INTERVENTION

Focus Mode

When overload is detected, NeuroFlow applies Focus Mode: greyscale filter, sidebars collapse, animations stop, font size increases. Instant calm.

body.high-load-mode → calm CSS
📊
REPORT

Neuro-Audit Report

After a session, generate a developer report: "This popup causes a 40% spike in AU04 stress for neurodivergent testers." Actionable, quantified accessibility insights.

POST /analyze → AU timeline JSON
Technology

The Double-Threat Stack

Two state-of-the-art computer vision libraries, each doing what it does best.

🎯

MediaPipe

Google's ML framework — Face Landmarker with 52 blendshape coefficients running at 30fps in the browser. Zero installation, zero data upload.

  • 🔸 JavaScript / WebAssembly
  • 🔸 GPU-accelerated
  • 🔸 Tracks browDownLeft, browDownRight
  • 🔸 Triggers real-time UI changes
🧪

Py-Feat

Python's scientific facial action coding library. Precisely measures FACS Action Units from video with research-grade accuracy.

  • 🔸 Python / Flask Server
  • 🔸 FACS AU04 (Brow Furrow)
  • 🔸 FACS AU45 (Blink Rate)
  • 🔸 Cognitive Load Score output
